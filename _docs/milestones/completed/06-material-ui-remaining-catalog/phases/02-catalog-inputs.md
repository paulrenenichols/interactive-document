# Phase: Catalog — inputs (material-ui-remaining-catalog)

Implement input components for `libs/material-ui`: Checkbox, Switch, Radio, Slider, Select, NumberField, ButtonGroup, ToggleButton, Rating, Autocomplete, FAB. Each uses theme tokens, supports `.dark`, and has a Storybook story.

---

## Scope

- **Inputs:** Checkbox (standard + indeterminate; optional label), Switch (toggle + optional label), Radio (radio group wrapper), Slider (continuous, discrete, range; value label), Select (dropdown; can use Popover), NumberField (numeric input with increment/decrement), ButtonGroup (shared styling), ToggleButton (on/off; exclusive and grouped), Rating (star-based; standard, half, custom icons), Autocomplete (combobox with search/filter; build on TextField + Menu/Popover), FAB (floating action button; small/regular/large, extended).
- Per exploration: export from `libs/material-ui/src/index.ts`, update README, add stories.
