# Planning — Stitch UI (MUI / Tailwind ↔ Stitch)

**Milestone:** [stitch-ui](../../../milestones/active/stitch-ui/) (`_docs/milestones/active/stitch-ui/`)

## Summary

Align the interactive-document frontend with Stitch **Interactive Document** exports: **light** frames are canonical for layout/IA; **dark** informs tokens and atmosphere via shared CSS variables and MUI. Implementation uses `@interactive-document/material-ui` and Tailwind v4 theme tokens in `libs/material-ui/src/theme.css`.
