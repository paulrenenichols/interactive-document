# Explorations

Informal research and ideation for future milestones. Each subfolder is named after a proposed feature set (e.g. `export-pdf`, `real-time-collaboration`) and can contain markdown files, screenshots, mockups, and other assets.

Explorations that have been turned into milestones are moved to [`completed/`](completed/).

To evaluate an exploration and discuss turning it into a milestone, use the prompt in [planning/setup/exploration-lifecycle.md](../setup/exploration-lifecycle.md).
