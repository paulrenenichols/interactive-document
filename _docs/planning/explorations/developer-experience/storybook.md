# Storybook

## Summary

Add Storybook to the repo for component development in isolation. The project does not currently have Storybook.

## Scope

- Install and configure Storybook for the frontend app (Next.js in `apps/frontend`).
- Set up so UI components (e.g. editor, viewer, charts, auth) can be developed and reviewed in isolation with different props and states.
- Align with existing stack: React, Next.js App Router, theme tokens (see [theme-rules](../../milestones/00-initial-milestones/definition/theme-rules.md) and [ui-rules](../../milestones/00-initial-milestones/definition/ui-rules.md)).

## Out of scope

- Changing production build or deployment; Storybook is a dev-time tool.
