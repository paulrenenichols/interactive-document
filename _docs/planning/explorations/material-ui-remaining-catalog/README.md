# Material UI remaining catalog

Created with the **docs-driven-dev** skill (v1.4.0).

This exploration scopes the **remaining** components and utilities for the `libs/material-ui` M3 component library. The library was started in the ux-and-stack-overhaul milestone (phase 03: library + 4 components; phase 03b: theme/transitions, utilities, and a first batch of catalog components). What exists today is documented in [progress 03](../../../progress/ux-and-stack-overhaul/03-material-component-library.md) and [progress 03b](../../../progress/ux-and-stack-overhaul/03b-material-ui-catalog-and-polish.md).

**Goal:** Define and prioritize the rest of the catalog so it can be implemented in a follow-up phase (e.g. a new milestone or 03c) or incrementally as the app needs each component.

Use [exploration-lifecycle.md](../../setup/exploration-lifecycle.md) to evaluate this exploration or turn it into a milestone.

## Structure

Feature sets are in `feature-sets/` (see [feature-sets/README.md](feature-sets/README.md)). Supporting materials are in [supporting-docs/](supporting-docs/) with a short index in [supporting-docs/README.md](supporting-docs/README.md).

## Scope

- **Already done (not in scope):** Button, TextField, Dialog, Menu/MenuItem, Fade, Grow, Portal, Modal, Popover, ClickAwayListener, useMediaQuery, CSSBaseline, Box, Stack, Typography, Paper, IconButton.
- **In scope:** All other components and layout primitives listed in the progress gap table — inputs (Autocomplete, ButtonGroup, Checkbox, FAB, NumberField, Radio, Rating, Select, Slider, Switch, ToggleButton), data display (Avatar, Badge, Chip, Divider, Icons, List/ListItem, Table, Tooltip), feedback (Alert, Backdrop, CircularProgress, LinearProgress, Skeleton, Snackbar), surface (Accordion, AppBar, Card), navigation (BottomNavigation, BottomSheet, Breadcrumbs, Drawer, Link, NavigationRail, Pagination, Stepper, Tabs), layout (Container, Grid, ImageList).

See [remaining-material-ui-catalog.md](feature-sets/remaining-material-ui-catalog.md) for the full list, dependency order, and references to the original exploration and progress docs.
