## Remaining Material UI catalog

Single feature set that defines all remaining components and layout primitives for the material-ui library (inputs, data display, feedback, surface, navigation, layout). Implement in dependency order; use theme tokens and Fade/Grow/Slide/Collapse where specified.

See [remaining-material-ui-catalog.md](remaining-material-ui-catalog.md).
